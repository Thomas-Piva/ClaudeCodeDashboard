Ho analizzato tutto il codebase. Seleziona le aree che vuoi approfondire.

🔴 Bug / Problemi
1

Reverse path conversion lossy

claudeDirNameToPath() converte tutti i - in \ — percorsi con spazi o caratteri non-ASCII vengono decodificati in modo sbagliato

2

localhost:3001 hard-coded

ProjectCard.jsx usa http://localhost:3001 in 4 punti distinti — non funziona se il backend è su porta diversa o host remoto

3

terminal-pids.json non si pulisce

Le voci in terminal-pids.json si accumulano indefinitamente — PID di processi morti rimangono, creando match falsi in "trova finestra"

4

Lettura intera sessione ogni 10s

readLastNLines() legge tutto il file .jsonl in memoria. Il periodic check ogni 10s rilegge TUTTI i progetti anche se inattivi

5

alert() / confirm() nativi

Dialoghi browser bloccanti in ProjectCard.jsx — rompono l'estetica Terminal Noir e bloccano il thread UI

6

Nessun rate limit su /terminal-windows

Ogni click "trova finestra" avvia un processo PowerShell da ~150 righe. Click rapidi possono accumulare processi PS paralleli

🟡 Miglioramenti UX
A

Toast notifications

Sostituire alert/confirm con notifiche a scomparsa in-app, coerenti col tema Terminal Noir

B

Ricerca globale (tutte le colonne)

Estendere la ricerca attuale (solo Inattivi) a tutte e 3 le colonne — utile con molti progetti

C

Ordinamento card

Ordina per nome, ultimo aggiornamento, durata sessione — attualmente l'ordine è casuale in base alla discovery

D

Modalità compatta

Card più dense per chi monitora molti progetti — toggle tra layout espanso e compatto

E

Azioni batch

"Segna tutti come controllati" per svuotare la colonna Check in un click quando si torna da una pausa

F

Keyboard shortcuts

Scorciatoie rapide: F per trova finestra, C per segna controllato, / per ricerca

🟢 Nuove Utility
I

Notifiche desktop

Web Notifications API quando un progetto passa a "Da Controllare" — avviso anche se la dashboard non è in primo piano

II

Statistiche per progetto

Tempo totale attivo oggi, numero sessioni, tool più usati — storico leggibile dai file .jsonl esistenti

III

Timer sessione live

Contatore "da quanto è attivo" aggiornato in tempo reale sulla card — visibilità immediata della durata

IV

Note / tag per progetto

Aggiungi una nota rapida a un progetto ("cliente: ACME", "in attesa feedback") — salvata in un file JSON locale

V

Raggruppamento per scan root

Mostra una sezione per ogni cartella radice (BIZ2017 / Progetti Pilota / ProgettiEgm) invece di un flat list

VI

Export storia sessione

Esporta la storia output di un progetto come testo/markdown — utile per documentare cosa è stato fatto
